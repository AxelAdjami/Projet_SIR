# Projet_SIR
Projet INSA Lyon : Conception d'un boîtier de diffusion vidéo sécurisé et non intrusif sur Raspberry Pi. 
Ce dépôt regroupe les travaux de recherche et de prototypage pour une solution de déport d'affichage sécurisé.
 Structure du projet 
* **`docs/`** : Etat de l'art, Rapport Eduroam WebRTC,Explication de WebRTC, Solution retenue, Vecteurs d'attaque
* **`prototypes/`** : Codes sources
* **`solution_avec_eduroam/`** : Version utilisant le réseau Wifi Eduroam.
* **`solution_sans_eduroam/`** : Version fonctionnant en réseau local AP.
